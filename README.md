# Dox — Document Vault PWA

Privacy-first family document vault for India.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:5173 — use **Continue with Google (demo)** for local-first mode (no Supabase required).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Dev server |
| `npm run build` | Production build |
| `npm test` | Vitest unit tests |
| `npm run preview` | Preview production build |

## Agent workflow

See [docs/AGENTIC_LOOP.md](docs/AGENTIC_LOOP.md) and `.cursor/agents/`.

## Environment

Copy `.env.example` to `.env` and set Supabase keys when connecting a backend.
